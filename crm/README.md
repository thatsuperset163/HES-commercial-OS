# Commercial Sales OS (source)

Vite/React app embedded in HQ at `/work/sales/`.

Edit here, then from the **repo root** run:

```bash
npm run sync:sales
```

That builds this app into `../public/work/sales/` for the Next.js HQ deploy (Vercel).

Do not deploy this folder alone on Vercel anymore — deploy the **HQ root** instead.
